# Salla Call Center Performance Dashboard

This project presents a comprehensive analysis of call center performance metrics and operational insights for Salla.  
The dashboard helps in monitoring efficiency, customer experience, and overall call handling performance.

---

## Key KPIs

- **Total Calls:** 1.7 Million  
- **Handling Rate:** 98.7%  
- **Average Speed of Answer (ASA):** 9 Seconds  
- **Abandon Rate:** 1.32%

## Project Objective

The goal of this dashboard is to provide clear visibility into call center operations, identify performance bottlenecks, and support data-driven decision making to improve customer service quality.

---

## Key Insights

- High handling rate indicates efficient call resolution process  
- Very low ASA shows strong operational responsiveness  
- Low abandon rate reflects good customer experience
